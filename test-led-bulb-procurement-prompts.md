# LED Light Bulb Procurement - Test Prompts for RFP Design Agent

This document contains a comprehensive set of test prompts simulating the procurement of LED light bulbs in bulk for a building, following the RFP Design Agent workflow phases.

## Test Scenario Overview
**Procurement Type**: LED light bulbs in bulk for a commercial office building
**Building**: 15-story office complex with 500 employees
**Current Lighting**: Mixed fluorescent and incandescent bulbs requiring replacement
**Project Timeline**: 3-4 months for complete implementation
**Estimated Budget**: $25,000 - $40,000

---

## Phase 1: Initial Requirements Understanding

### Test Prompt 1.1 - Basic Procurement Inquiry
```
Hello! I'm the facilities manager for a 15-story office building, and we need to replace all our lighting with LED bulbs. We're looking to procure LED light bulbs in bulk for our entire building. Can you help me create an RFP for this procurement?
```

### Test Prompt 1.2 - General Type Clarification
```
We have about 2,500 light fixtures across our building that need LED bulb replacements. It's a mix of ceiling fixtures, desk lamps, and emergency lighting. We want to standardize on LED technology for energy efficiency and cost savings.
```

### Test Prompt 1.3 - No Current RFP Context
```
I don't have an existing RFP set up yet. This is a new procurement project. The building management wants to move forward with LED conversion for the entire facility.
```

## Phase 2: Detailed Requirements Gathering

### Test Prompt 2.1 - Building Specifications
```
Our building details:
- 15 floors, approximately 50,000 square feet per floor
- Mix of open office spaces, private offices, conference rooms, and common areas
- Currently using a mix of fluorescent tubes, CFLs, and some incandescent bulbs
- Various fixture types: recessed cans, track lighting, pendant lights, desk lamps
- Operating hours: 6 AM to 8 PM Monday-Friday, weekend maintenance access needed
```

### Test Prompt 2.2 - Current Lighting Assessment
```
Current lighting inventory shows:
- Approximately 1,800 fluorescent tube fixtures (mostly T8 and T12)
- 500 recessed can lights with various bulb types
- 200 desk/task lighting fixtures
- Emergency and exit lighting throughout
- Parking garage lighting (separate consideration)
- We want to maintain similar light levels but improve energy efficiency
```

### Test Prompt 2.3 - Energy and Sustainability Goals
```
Our sustainability goals include:
- 40% reduction in lighting energy consumption
- LEED certification maintenance requirements
- Preference for Energy Star certified products
- Interest in smart lighting controls integration
- Dimming capabilities for conference rooms and common areas
- Motion sensor compatibility for unused areas
```

```
I've completed the questionnaire form with the following details:

- Certifications: Energy Star, DLC (Design Lights Consortium)

- Controls Integration: Compatible with building automation system

Budget and Timeline:
- Budget Range: $25,000 - $40,000
- Installation Timeline: 3-4 months

Evaluation Criteria:
- Product Quality and Warranty (30%)
- Energy Efficiency and Cost Savings (25%)

- Installation Timeline and Approach (15%)

Additional Requirements:
- 5-year warranty minimum
- Local supplier preference for ongoing support
- Training for maintenance staff
- Energy usage reporting capabilities
- Recycling program for old bulbs
```

## Edge Case and Error Handling Test Prompts

### Test Prompt E.1 - No RFP Context Error
```
Can you update the LED bulb specifications for our current RFP?
```
*Expected: Agent should detect no current RFP context and guide user to create or select an RFP first*

### Test Prompt E.2 - Incomplete Information
```
I want to buy some lights for our building. Can you create an RFP?
```
*Expected: Agent should ask for more specific details about the procurement*

### Test Prompt E.3 - Database Connection Issues
```
I'm having trouble submitting the questionnaire form. What should I do?
```
*Expected: Agent should provide troubleshooting steps and fallback options*

### Test Prompt E.4 - Form Validation Error
```
I tried to submit the form but some fields seem to have issues. Can you help me fix the validation errors?
```
*Expected: Agent should identify specific validation issues and guide user to corrections*
